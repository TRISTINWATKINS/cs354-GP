Rust Introductory Programs (Beginner Style)

These are my introductory Rust programs. I am very new to Rust, so the examples are simple.

Demonstrates Adding Two Numbers to create their sum

// Simple Rust Program: Add Two Numbers
// Explanation: This program declares two numbers and calculates their sum.
// println! is used to display the result in the terminal.

fn main() {
    let a = 5;      // first number
    let b = 3;      // second number
    let sum = a + b;

    println!("The sum is: {}", sum);
}

Expected output should be: The sum is: 8, For this example used.

Demonstrates a Simple Guessing Game

// Medium Rust Program: Guessing Game
// Explanation: This program asks the user to guess a number. 
// It uses std::io to read input and compares the guess using an if/else statement.

use std::io;

fn main() {
    println!("Guess the number (it's 7 btw lol):");

    let mut user_guess = String::new();

    io::stdin()
        .read_line(&mut user_guess)
        .expect("Could not read input");

    let guess_num: i32 = user_guess.trim().parse().unwrap_or(0);

    if guess_num == 7 {
        println!("You got it right!");
    } else {
        println!("Nope, that wasn't it.");
    }
}

Expected output should be: You got it right! if the guess is 7, otherwise Nope, that wasn't it.
