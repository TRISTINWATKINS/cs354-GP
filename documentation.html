2. Language Specification

Rust does not have a single formal language specification like C or Java. Instead, the Rust Reference and compiler behavior define the language. Important sections include memory model, ownership rules, pattern matching syntax, and type system behaviors.

Verify Installation

4. Compiling and Running Rust Programs

Without Cargo (single-file):

rustc program.rs
./program

With Cargo (recommended):

cargo new myproject
cd myproject
cargo run

5. Rust Development Tools

  • rustc: The Rust compiler
  • Cargo: Build system and package manager
  • rustfmt: Auto-formatter
  • clippy: Linter / static analysis
  • crates.io: Library/package repository
  • docs.rs: Documentation for all published crates
